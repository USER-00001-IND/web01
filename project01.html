📘 Library Management System (C++ Project)

👨‍💻 Project Overview

This is a simple command-line based Library Management System written in C++. It allows users to add, view, search, and delete books. All data is stored in a file called library.txt, which acts as the system's database.

⚙️ Working Mechanism

  • Data Structure: Uses a struct Book to store book information: ID, title, and author.
  • File Handling: Book data is stored in a text file using ofstream and ifstream.
  • Delimiter: Book fields are separated by the | character for easy parsing.
  • Functions:
    • addBook() – Adds a new book to the file (after checking for duplicate IDs).
    • displayBooks() – Reads and displays all books from the file.
    • searchBook() – Searches for a book by ID.
    • deleteBook() – Deletes a book by ID by copying all other books to a temporary file.
  • Main Menu: A loop in main() displays options for user interaction.

📂 File Structure

Each line in library.txt contains a book in this format:

101|The Great Gatsby|F. Scott Fitzgerald

🔐 Input Validation

The program checks for:

  • Non-numeric IDs
  • Duplicate book IDs
  • Empty files when displaying books

🎯 Features

  • Simple and user-friendly interface
  • Persistent file-based storage
  • Easy to modify and extend (e.g., edit books, sort list)
  • Basic error handling for invalid input
